# Habit Tracker App

This is a simple habit tracker React application where users can input their habits, including what to do, where to do it, why to do it, and when to do it.

## Features
- Add new habits with details (what to do, where to do it, why to do it, and when to do it).
- View a list of added habits.
- Delete habits from the list.

## Usage
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server using `npm start`.
4. Open the application in your browser.
